# Tokenizer Web App

A simple web application that counts characters and tokens from user input text. Built using HTML, CSS, and JavaScript.

## Features
- Counts total characters
- Counts tokens using 4-character chunk method
- Clear button to reset the text area
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Type or paste your text into the textarea
2. Click the "Count" button to calculate characters and token count
3. Click "Clear" to reset everything

## Project Structure
index.html
style.css
script.js

## Example
Input:
Hello world

Output:
Characters: 11
Tokens: 3

## Run Locally
- Download or clone the project
- Open index.html in any browser
